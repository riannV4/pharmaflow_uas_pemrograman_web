🎯 CUSTOMIZATION GUIDE
═════════════════════════════════════════════════════════════════════════════

  Change Colors:
    → Edit app/globals.css
    → Update CSS variables

  Adjust Animation Speed:
    → Edit duration in component files
    → Speed format: duration: 0.6 (in seconds)

  Change Font:
    → Edit app/layout.tsx
    → Import different Google Font

  Modify Marquee Speed:
    → Edit tech-marquee-section.tsx
    → Change speed prop (40 = 40 seconds per cycle)

  Add New Section:
    → Create component in components/landing/
    → Import in app/page.tsx
    → Add to component tree

💡 PRO TIPS

  1. Use clamp() for fluid typography
  2. GPU-accelerate with transform + opacity only
  3. Add viewport={{ once: true }} to scroll triggers
  4. Test on real mobile devices
  5. Use Chrome DevTools Performance tab
  6. Keep animations < 1 second for best feel
  7. Use linear ease for continuous animations
  8. Pause animations on reduced-motion preference
